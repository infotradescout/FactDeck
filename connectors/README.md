# Connectors

Place structured decision-domain data connectors here.
Keep ingestion explicit and auditable by source.

Current connector datasets:
- `data/demographics.json`
- `data/search_signals.json`
- `data/cost_pressures.json`
- `data/regulatory_flags.json`

Use from CLI with:
- `--use-connectors`
- `--evidence-dir connectors/data`
